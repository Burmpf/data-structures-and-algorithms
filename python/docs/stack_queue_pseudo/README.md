# Stack and queue pseudo
## Challenge
use stacks to pseudo run as a queue

## Approach & Efficiency
- These function as following:
  - time- O(1)
  - space- O(n)
  
## Methods
Dequeue- removes(pop) an item and places it (push) in a new stack
enqueue- same as dequeue
